INFORMATION VISUALISATION

Information visualization is the practice of representing data and information in a visual or graphical form to facilitate understanding, exploration, and communication. It involves transforming complex data sets into intuitive visual representations that allow viewers to extract meaningful insights and patterns.


Within the domain of information visualization, extensive efforts were dedicated to both experimentation and exploration of a diverse range of tools and techniques. Notable tools such as d3.js, prominent Python libraries, and the versatile platform of Tableau were extensively utilized in order to achieve comprehensive insights. This pursuit involved a meticulous examination of various visualization methodologies, encompassing advanced concepts and intricate frameworks.


  • d3.js:D3.js (Data-Driven Documents) is a JavaScript library for creating interactive data visualizations on the web. Observable HQ is an online platform that combines D3.js with a collaborative environment for creating and sharing interactive visualizations. It offers a notebook-style interface, real-time updates, and customization options, enabling users to create dynamic visualizations and collaborate with others. Together, D3.js and Observable HQ empower users to explore and present data in engaging ways while fostering collaboration within the data visualization community.
  • Python libraries: Python libraries for data visualization, such as Matplotlib, Seaborn, Plotly, and Bokeh, offer a wide range of capabilities to create visually compelling and informative graphics.
    Matplotlib is a versatile library that provides a comprehensive set of functions for generating various types of plots, charts, and graphs.
    Seaborn, built on top of Matplotlib, focuses on statistical visualization. It provides a high-level interface and pre-defined themes that simplify the process of creating aesthetically pleasing statistical plots
    Plotly is known for its interactive visualizations and rich charting capabilities. It allows users to create dynamic and interactive plots that respond to user interactions like zooming, panning, and hovering
  • Tableau: Tableau is a widely used data visualization and business intelligence software that enables users to analyze, visualize, and share data in an intuitive and interactive manner. With its user-friendly interface, drag-and-drop functionality, and diverse visualization options, Tableau allows users to create compelling charts, graphs, and dashboards. It offers advanced analytics features, supports data blending, and provides capabilities for geographic mapping and forecasting. Tableau facilitates interactive exploration of data, promotes collaboration through sharing options, and empowers users to derive valuable insights and make data-driven decisions.
